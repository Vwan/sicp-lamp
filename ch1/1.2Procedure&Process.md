#### 实现方式 1：回归

procedure factorial 调用自身

python:

```
def factorial(n):
	if n == 1:
		return 1
	else: 
		return n * factorial(n-1)

factorial(5)
```

(define (factorial n)
  (if (= n 1) 1

#### 实现方式2：循环迭代

python:

```
def factorial(n):
	result = 1;
	for i in range(1, n+1):
		result *= i
	return result

factorial(5)
```

#### 实现方式3：迭代

python

```
def factorial(n):
	def iter(result, count, max):
		if count > max:
			return result
		else:
			return iter(result * count, count + 1, max)
	return iter(1, 1, n)	

print(factorial(5))
```

lisp:

```
(define (factorial n)
  (define (iter result count max)
    (if (> count max) result
	(iter (* count result) (+ count 1) max)
     )
    )
  (iter 1 1 n)
  )

(factorial 5)
```
